How To build A Website

Website building is actually a lot easier to do with the help of a pre-made template. For beginners, like myself, w3schools.com has been one of the best guides in building a website. You can learn how to code in different programs like HTML, CSS, Javascript, and so many more. They show you how to write in the correct format and how to tell the computer a command using the different code. This teaches you to be creative and adventurous. They give you the tarter but of course you have to use your imagination and create your own site. If you are using a Microsoft Computer, Notepad ++ is a perfect tool for writing the code for your index. If you are a Mac Computer user, like myself, then Sublime is the perfect tool to code. They are both easy to use and write on and all you have to do once you code your work is save it and just refresh your Internet page; viola your work is officially published.
